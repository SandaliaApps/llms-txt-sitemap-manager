== Frequently Asked Questions ==

= What is LLMs.txt? =
LLMs.txt is a simple, structured file that lists public URLs from your site to help AI models like ChatGPT understand and discover your content.  

= Does it integrate with SEO plugins? =
Yes! LLMs.txt Sitemap Manager works seamlessly with Yoast SEO, Rank Math, and AIOSEO, automatically excluding content marked as noindex or nofollow.  

= Can I customize which posts appear? =
Absolutely! You can select post types, order them, and configure how frequently the file updates.
